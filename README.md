Два проекта, посвящённые задаче фильтрации данных.
Общая идея для всех проектов, инкапсульировать фильтры в отдельные сущности.
Filtering - фильтрующая система, основанная в целом на ОО-принципах
FunWithFilter - система более ориентированная на принципы ФП + пример использования с WPF
UnitTestSearch - набросок unit-тестов для Filtering
